# opengl_disputes
Host warious OpenGL examples.

# Create a new dispute (a.k.a. project)
```
opengl_disputes % make new_dispute
opengl_dispute_name [the_box]: <HIT ENTER>
opengl_disputes % cd the_box
the_box % mkdir build
the_box % cd build
build % cmake ..
build % make the_box
build % ./the_box
```
